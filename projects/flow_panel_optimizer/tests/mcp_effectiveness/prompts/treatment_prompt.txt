{marker_list}

## Available Tools:

You have access to these MCP tools for panel design:

1. **analyze_panel(fluorophores)**: Analyzes a list of fluorophores and returns:
   - Similarity matrix (pairwise cosine similarity)
   - Complexity index (overall interference score, lower is better)
   - List of problematic pairs (similarity > 0.90)

2. **check_compatibility(candidate, existing_panel)**: Checks if a candidate fluorophore is compatible with existing selections:
   - Returns max similarity to existing fluorophores
   - Recommendation: "safe" / "caution" / "avoid"
   - Lists any conflicts

3. **suggest_fluorophores(existing_panel, expression_level)**: Suggests best fluorophores given current panel:
   - Returns ranked list of compatible options
   - Considers both spectral separation and brightness

## Instructions:

Build the panel iteratively using the tools:

1. Start with the highest-expression markers (more fluorophore options)
2. For each marker:
   a. Use `suggest_fluorophores` to get candidates
   b. Use `check_compatibility` to verify the best option
   c. Add the selected fluorophore to your panel
3. After completing the panel, use `analyze_panel` to verify quality
4. If complexity index is high or critical pairs exist, revise selections
